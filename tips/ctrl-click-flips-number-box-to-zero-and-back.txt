TITLE	Ctrl-click flips Number box to zero and back
DETAIL	When in Run mode, click a Number box with the Ctrl or Alt key held down to set its value to zero immediately.
DETAIL	
DETAIL	Ctrl-click or Alt-click again, and the previous value is restored.
